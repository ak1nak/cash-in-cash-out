# Commission Fee Calculator

This Node.js application calculates commission fees for cash-in and cash-out operations based on given JSON data.

## Getting Started

### Prerequisites

You need Node.js and npm installed on your machine to run this application. You can verify the installation with:

```bash
node -v
npm -v
```

## Run the Application

node src/app.js <path-to-input-file>

## Run tests

npm test
